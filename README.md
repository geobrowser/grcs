# Geo Request for Comments (GRCs)

This repository contains the formal specifications for the [Geo protocol](https://geobrowser.io).

## GRCs

| Number | Title | Status |
|--------|-------|--------|
| [GRC-20](grcs/grc-0020.md) | Knowledge Graph | Final |

## Contributing

GRCs follow a proposal lifecycle: **Draft** > **Review** > **Final**.

To propose a new GRC, open a pull request with a new file in the `grcs/` directory following the naming convention `grc-NNNN.md`.
